# Google Search Frontend Clone

A beginner web development project that recreates the core interface of Google Search using **HTML** and **CSS**. The project was built as part of my learning journey to understand how web pages are structured, styled, and connected through forms.

## Features

* Google Search page
* Google Image Search page
* Google Advanced Search page
* Functional navigation between all pages
* Working search forms that submit queries to Google's search engine

## Technologies Used

* HTML5
* CSS3

## Running the Project

1. Download or clone this repository.
2. Ensure all project files remain in the same folder.
3. Open `index.html` in any modern web browser.

No installation or additional dependencies are required.

## Purpose

The goal of this project was to gain hands-on experience with:

* HTML document structure
* Forms and form submission
* Links and navigation
* CSS styling and layout
* Flexbox
* Building a multi-page website

This project focuses on functionality first, followed by styling to closely resemble the original Google Search interface.

This reads more like a professional GitHub README while still making it clear that it's a learning project. It also gives visitors enough context to understand what they'll see without overselling the project.
